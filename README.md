
<h1>JOB PLATFORM APP SNAPSHOTS</h1>


Job Platform Application Developed using JSP, Servlet, JDBC and MySQL Database.

## Following features
- Add Jobs
- Edit Jobs
- List Jobs
- Delete Jobs
- Apply Jobs
- View Applied Jobs
- Login Model
- Registration Model
- Forgot Password Model
- Search Model
- Admin Module
- User Module
- Responsive layout

## Technology stack
- Java 8+
- JSP, HTML5, CSS3, BOOTSTRAP4 and Servlet 3+
- JDBC 4+
- MySQL database 8+
- Eclipse IDE 2021-09
- Tomcat server 9+
